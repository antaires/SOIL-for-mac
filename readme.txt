How go get SOIL for mac:
Download the mac header. (the SOIL.h file in this directory)

In a terminal window, from the directory where you downloaded the header file:

$ sudo mkdir /usr/local/include/SOIL

$ sudo cp SOIL.h /usr/local/include/SOIL/

Download the mac library (the libSOIL.a in this directory)

In a terminal window, from the directory where you downloaded the library:

$ sudo cp libSOIL.a /usr/local/lib/
